ICU is released under a nonrestrictive open source license
https://www.unicode.org/copyright.html#License

1. Navigate to the offical packaged version of the ICU4C source code - USE AT LEAST VERSION 69
   http://site.icu-project.org/download

2. Download the offical packaged version off of github under (latest) releases. Make sure you get the one that matches your build environment.
   
3. Extract the contents into .../HarmonyEngine/Engine/extras/ICU - DO NOT INCLUDE THE ROOT FOLDER

3. Create new builds that enable ICU by choosing the option in the build settings dialog within the Harmony Editor
